# 俗幻辞典携帯用レイアウト設定
## 本ツールについて
本ツールは、[俗アルカ辞典](https://mindsc.ape.jp/klel/)を携帯電話でも使いやすいレイアウトにするためのスクリプトです。本ツールは非公式のものであり、作成者は俗アルカおよび本家アルカのいずれとも関係を有していません。
## 各ファイルの動作について
### [main.js](../main.js)
以下の動作を含みます。
- 携帯で文字が小さくなり過ぎないようにする
- 検索部分と結果表示部分を縦並びにする
- 検索部分が縦長になり過ぎないよう、一部部品を横並びにする
- 検索後に検索ワード入力欄にフォーカスが当たるのを防ぐ（ソフトウェアキーボードが表示されないようにするため）

### [with_option.js](../with_option.js)
main.jsの内容に加え、以下の動作を含みます。
- 検索ワードが英数字のみのときは見出し語検索モードを、英数字以外を含むときは訳語検索モードを自動で選択する（ただし、用例検索または全文検索のモードが選択されているときを除く）

## 使い方
### 登録
携帯のブラウザで、以下の通りブックマークを作成します。
- タイトル：任意
- URL: main.jsまたはwith_option.jsの中身をすべてコピー&ペースト

### 使用
1. 俗幻辞典を開きます。
2. 作成したブックマークを開きます。通常、ブックマークを開くと登録したサイトに遷移しますが、本ツールのようにスクリプトを登録した場合は、開いているページに当該スクリプトが実行されます。
